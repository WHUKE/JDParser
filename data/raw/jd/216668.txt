消费金融高级运维工程师（Java）
广东省
研发类

业务线：京东科技
公司：京东
发布时间：2026-04-27

岗位描述
1、负责海外消费贷核心系统的应用运维与稳定性保障，支撑授信、额度、定价、借还款、账务、资金路由等关键链路稳定运行。
2、负责 Java 微服务体系下的服务部署、配置管理、发布变更、版本回滚、日志分析、故障排查、性能跟踪和风险治理。
3、建设和优化监控告警、巡检、应急预案、故障复盘等机制，提升问题发现、定位和恢复效率。
4、参与核心系统高可用建设，推动限流、降级、熔断、灰度发布、容量评估、压测演练、容灾演练等工作落地。
5、负责 MySQL、Redis、Kafka、RocketMQ 等组件在业务系统中的运行保障、问题排查和容量治理。
6、与研发、产品、风控、运营、支付、资金等团队协作，参与重大业务上线、版本发布和核心链路变更保障。
7、推动运维自动化、标准化和工具化建设，合理使用 AI 辅助排障、日志分析、脚本生成和文档沉淀，提升运维效率。

任职要求
1、本科及以上学历，计算机、软件工程、信息系统等相关专业优先。
2、3-5 年应用运维、SRE、运维开发或 Java 系统运维相关经验，有复杂线上系统稳定性保障经验。
3、熟悉 Java 应用运行机制，能够分析 JVM、GC、线程池、连接池、慢接口、异常堆栈等常见问题。
4、熟悉 Spring Boot、Spring Cloud、Linux、Shell/Python，以及 MySQL、Redis、Kafka、RocketMQ 等常见技术栈。
5、具备分布式系统运维经验，理解高可用、容灾、监控告警、灰度发布、容量规划、故障演练等稳定性建设方法。
6、必须具备金融支付系统、交易系统、账务系统或资金类系统的运维/稳定性保障经验，理解金融系统对一致性、连续性和故障恢复的高要求。
7、有金融、消费金融、信贷、支付、账务、借还款等业务系统运维经验者优先。
8、具备较强的问题分析、风险识别和跨团队协作能力，能在复杂线上环境中快速定位并推动问题解决。
9、具备英文技术文档阅读能力；英语口语流利、具备 AI 辅助运维实践经验者优先。
10、符合京东价值观：客户为先、创新、拼搏、担当、感恩、诚信。
