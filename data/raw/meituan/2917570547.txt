工作职责
1、AI Coding 能力的发展，让技术语言不再成为壁垒，团队判断未来 1-3 年全栈将会成为趋势
2、AI 应用需要快速迭代和探索，全栈闭环可以让事情更容易发生，不用陷入资源协调的损耗中
3、团队希望能找到更多认可此判断的同学加入，在核心稳定业务中共同转型岗位要做什么：
4、【业务层】① 负责美团到店餐饮业务商家及销售方向，结合已有系统能力，完成在销售和供应链方向 Agent 探索与应用。② 深度与业务、产品、运营合作，高频共创和下一线，明确 PTMF。
5、【产品层】① 确保 AI 技术融合到商家及销售智能化产品流程中，保证 GUI 和 NUI 无缝衔接，探索 Generative UI。② 建设智能问答（知识+工具）、智能外呼、数据分析等应用层能力。③ 通过感知和理解用户的目标、资源、行为，完成“人用工具 → 工具找人”的主动服务型产品能力转变。
6、【系统层】① 设计、实现和维护先进的 Agentic System，结合 Agentic RAG、Cognitive Architecture、Agentic Workflow 等前沿技术提升准确率和性能，降低系统成本。② 解决在多模态模型、多 Agent 场景下的可扩展性问题，创造一流的工程体验。③ 完成企业存量 Tools 的低成本集成，结合 CodeAgent 和 SpecCoding 模式，让用户能够自建 Skill。
7、【数据层】① 保证底层数据口径统一，面向 Agent 友好。② 提供方便的企业数据集成和查询能力。③ 低成本的完成知识库生产、管理、消费流程，保证知识无过时、冲突、冗余。④ 完善基于测评驱动的 Agent 生产流程，保证测评集的时效性，快速评估 LLM 和 Agent 效果
8、【模型层】① 周期性进行 SOTA 模型测评，并根据不同场景和模型特点选择高性价比模型。② 协同训练师团队完成模型所需训练数据的生产、标注任务。③ 完成领域场景模型的 SFT、DPO 工作。④ 与算法团队共同探索 GRPO、PPO 等模型训练工作。

任职要求
1、曾直接与客户合作，深入了解客户需求，并熟练使用他们使用的工具。
2、在减轻风险和构建可靠、可扩展和高性能系统方面有良好的判断力。
3、出色的沟通技巧，能够有效解释你决策背后的“为什么”。
4、对 Agentic System 有扎实的理解，有使用 LangChain、LangGraph 等框架的经验。
5、AI Coding 出码采纳率和 AI 代码占比等关键指标上表现出色（团队当前采纳率 75%+，AI 代码占比 60%+），有 Spec Coding 实践经验
